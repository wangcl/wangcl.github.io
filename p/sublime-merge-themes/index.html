How to

generally, a sublime merge theme contains:

  • theme file

    1. {THEME}.sublime-theme

  • color scheme files

    1. {THEME}.sublime-color-scheme

    2. Commit Message - {THEME}.sublime-settings

    3. Diff - {THEME}.sublime-settings

    4. File Mode - {THEME}.sublime-settings

    5. Widget - {THEME}.sublime-settings

  1. copy those files to：

    {SUBLIME_MERGE_HOME}/Data/Packages/User

  2. modify settings：

    {SUBLIME_MERGE_HOME}/Data/Packages/User/Preferences.sublime-settings

    {
    "theme": "{THEME}.sublime-theme",
    "color_scheme": "{THEME}.sublime-color-scheme",
}
